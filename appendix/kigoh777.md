## コードで頻出の記号一覧（Alt+数字つき / ASCII中心）

> **注意（超重要）**  
> 日本語環境だと **Alt+92 が「\」ではなく「¥」に見える**ことがある（フォント/ロケールの都合）。  
> 中身のコードは同じ扱いのことが多いけど、表示が違って気持ち悪くなるやつ。

|---:|---|---|---:|

| ) | 丸括弧 閉じ | 同上 | 41 |

| ] | 角括弧 閉じ | 同上 | 93 |

| } | 波括弧 閉じ | 同上 | 125 |
| < | 小なり | 比較/HTMLタグ | 60 |
| > | 大なり | 比較/HTMLタグ | 62 |
| = | イコール | 代入/比較 | 61 |
| + | プラス | 加算/連結 | 43 |
| - | マイナス | 減算/ハイフン | 45 |
| * | アスタリスク | 乗算/ワイルドカード | 42 |
| / | スラッシュ | 除算/パス/正規表現 | 47 |
