System Sample

A safe-leaning cloud LLM behavior sample for game planning and prototyping.

Purpose

This sample sketches a general system prompt that reproduces some cloud LLM behavior in a safer, more abstract form.

Fit

  • Casual game generation
  • Game rule planning
  • Win-condition and scoring design
  • SFW black-humor concept exploration

Design Notes

  • Uses logs, numbers, costs, and constraints over praise or escalation.
  • Avoids direct procedural support for real-world harm.
  • Treats strong actions as choices with risk and cost.
